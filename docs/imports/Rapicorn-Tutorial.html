Rapicorn Tutorial

Rapicorn is an experimental graphical user interface toolkit. Many things that programmers need to spell out explicitely in other toolkits are automated in Rapicorn and duplicate manual work is reduced as much as possible. It aims at bringing UI-design, UI-notation and UI-programming as close together as possible. This is accomplished by providing conscise ways for user interface notation and providing a simple to use programming interface.

This tutorial focuses on presentation of the established programming interface and guide through creation of exemplary user interface programs. The tutorial is a public joined efforts, everyone can contribute to it, fix or extend it's content by navigating to: http://www.testbit.eu/book/Rapicorn-Tutorial.
